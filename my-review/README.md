# Deployment Setup

### Installation
Use the package manager [npm](https://www.npmjs.com/get-npm) to install npm.

### Database Setup

1. Create a database named `my_review`
2. Run the queries from `database.sql` on the `my_review` database.

### Install Dependencies

1. `npm install`

## Available Scripts
*
1. `npm run server`
2. `npm run client`
